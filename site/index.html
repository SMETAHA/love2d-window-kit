Quick start

Copy three files. Require what you need.

Start with WindowManager.lua. Add WindowStack.lua for multiple viewports and SystemWindow.lua for the OS window. 

local WindowManager = require("WindowManager")

local map = WindowManager.new({
  contentWidth = 4200,
  contentHeight = 2800,
  inertia = true,
  input = {
    trackpad = { smooth = true },
    touchscreen = { doubleTap = true }
  }
})
