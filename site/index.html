Three modules, no dependencies

Use only the pieces you need.

WindowManager owns one viewport. WindowStack handles layers, focus, capture and modal routing. SystemWindow configures the real OS window. 

local WindowManager = require("WindowManager")

local map = WindowManager.new({
  contentWidth = 4200,
  contentHeight = 2800,
  inertia = true,
  minZoom = 0.35,
  maxZoom = 3
})

map:centerOn(2100, 740, {
  duration = 0.4,
  easing = "inOutQuad"
})
